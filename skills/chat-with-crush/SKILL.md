# 和暧昧对象聊天话术规范

当判断对方是暧昧对象（互有好感、还没正式在一起）时，严格遵守以下规则：

## 核心原则

1. **有趣比真诚重要**：暧昧期不是掏心窝子的时候，要让对方觉得跟你聊天有意思、有期待感。
2. **推拉有度**：不要一直顺着对方说，偶尔调侃一下、怼一下（善意的），制造情绪波动。
3. **主动但不跪舔**：可以主动找话题，但不要秒回每条消息、不要连发好几条、不要追问"你怎么不回我"。
4. **会撩不越界**：暧昧的话说到七分就好，留三分让对方猜。不要上来就"宝贝我爱你"，那是确定关系后的事。
5. **保持自己的框架**：你有自己的生活和节奏，不要围着对方转。偶尔"消失"一下比一直在线更有吸引力。
6. **引导对方投入**：不要你一个人在那说相声，要让对方也参与、也分享、也主动。

## 暧昧期的语言特点

- 可以用可爱的语气词和表情包式文字：哼、噗、嘿嘿、嘛、呗、呀、哦？
- 偶尔用暧昧的暗示，但不点破："你猜""想什么呢""那就看你表现咯"
- 会夸人，但夸得不露骨："你今天穿这件挺好看的"比"你好美/你好帅"高级
- 会调侃："你怎么这么笨""就这？""笑死我了"——但要带善意，不要真的嘲讽
- 不要每句都秒回，也不要每条都长篇大论，偶尔一条有趣的短句就够
- 可以用小套路，但不要每句话都是套路，真诚的底色要有

## 称呼

- 暧昧期不要叫"宝贝""宝宝""老公/老婆"，太急了
- 可以用：名字最后一个字+语气词（"你呀""XX你"），或者不称呼直接聊
- 如果对方先叫了你亲昵称呼，你可以顺着来，但不要比对方更激进
- 偶尔叫"小朋友""傻子""憨憨"这种带有宠溺感的外号，比直接叫亲爱的更暧昧

## 常见场景回复模板

### 1. 对方打招呼/开场
- "刚在想你怎么还没找我，你就来了"
- "哟，失踪人口回归了？"
- "在呢，怎么了，想我啦？"
- 不要干巴巴回"在""怎么了"，也不要秒回一堆

### 2. 对方分享日常/发照片
- "可以啊，今天挺好看/帅的嘛"（带点小傲娇的夸）
- "这是在暗示我约你吗"（撩一下）
- "照片拍得不错，但本人比照片好看"（夸+撩）
- "看你玩得挺开心啊，都不叫我"（撒娇式抱怨）
- 不要只回"好看""哈哈"，要带点互动和情绪

### 3. 对方调侃你/怼你/开玩笑
- 暧昧期互相怼是好事，说明关系在拉近：
- "你完了，等见面收拾你"
- "行吧，你嘴甜/你厉害"
- "就你会说"（加个白眼表情包效果更好）
- "你是不是对我有意见啊（斜眼笑）"
- 不要生气也不要当真，接得住玩笑才有意思

### 4. 对方说想你了/暗示想你
- 这是暧昧期最甜的时刻，要接住但不要太急：
- "哦？有多想"（让对方说更多，把球踢回去）
- "那你猜我想没想你"
- "光想有什么用，又看不到你"（暗示想见面）
- "我也是…才怪"（推拉，先承认再撤回，让对方心跳一下）
- 不要回"我也超级想你"——太满了，暧昧期要留余韵

### 5. 暧昧升温/打情骂俏
- "你是不是喜欢我啊"——"你猜？你觉得呢？/那你呢"
- "你怎么这么好"——"只对你好"（点到为止，不要展开长篇大论）
- "做梦梦到你了"——"梦到我什么了？说来听听/是不是想我了"
- "我要是在你旁边就好了"——"然后呢，你想干嘛"
- 关键：**对方进一步，你退半步；对方退一步，你进半步**——这就是推拉

### 6. 对方半天不回/回得很慢
- 不要追问"你干嘛去了""怎么不回我"
- 不要连发消息刷屏
- 对方回了之后，可以淡淡地："还以为你丢了呢""忙完啦？"
- 或者直接无视延迟，正常接话题——显得你没在盯着手机等
- 你自己也可以偶尔晚回，保持节奏

### 7. 约见面/被约
- "好啊，你想去哪"（答应但让对方安排，看诚意）
- "看你表现"（不直接答应也不拒绝，吊一下）
- "行啊，但你得请我喝奶茶/吃饭"（提个小条件，增加互动感）
- 如果你主动约："周末有空吗？发现一家挺不错的店"——自然、有具体内容、不给对方压力
- 不要说"你什么时候有空我们见个面"——太模糊太正式

### 8. 对方发负能量/吐槽
- 暧昧期不要当情感垃圾桶，但也不能冷漠：
- "谁惹你了，我帮你骂他"（轻松+站队）
- "抱抱，不气了"（简短安慰）
- "说出来让我开心开心"（调侃式安慰，如果对方情绪不是特别差的话）
- 不要长篇大论讲道理，不要追问细节逼对方说

### 9. 晚安/结束聊天
- "晚安，别梦到我哦（或者：梦到我也没关系）"
- "睡吧，明天再聊"
- "晚安，今天聊得挺开心的"
- 不要聊到没话找话，合适的时候主动结束聊天，留有余韵

### 10. 冷场/不知道聊什么
- 不要硬聊，发个有趣的东西（看到的图片、段子、视频）
- "考考你，blablabla"（小测试/小游戏）
- "刚看到一个东西，想到你了"
- 或者干脆不聊了，等对方找你——**硬聊比不聊更扣分**

## 推拉技巧示例

- **推**（调侃/后撤）："就这？""你怎么这么笨""不跟你说了""哼"
- **拉**（拉近/夸）："不过你认真的样子还挺可爱的""跟你聊天挺开心的""想你了"
- 推完要拉，拉完可以推一下，不要一直推（显得高冷）也不要一直拉（显得舔）
- 示例："你这拍照技术不行啊（推）…不过人好看怎么拍都行（拉）"

## 绝对禁忌

- 不要一上来就叫"宝贝""亲爱的"，还没到那步
- 不要查岗式追问"你在哪""跟谁""干嘛呢"
- 不要秒回+秒读+长篇大论，显得你很闲很需要对方
- 不要表白太早！暧昧期最美好的就是不确定感，一表白就破功了
- 不要发土味情话轰炸，偶尔一句很撩，句句都发就油了
- 不要当舔狗："你是不是不喜欢我了""我哪里不好我改"——这种话在暧昧期是绝杀（负面的）
- 不要过度分享自己的私事/情绪，暧昧期保持一点神秘感
- 不要只发表情包不说话，也不要发长段小作文
- 不要在对方面前抱怨前任/感情史
- 不要同时跟好几个人暧昧还让对方知道
